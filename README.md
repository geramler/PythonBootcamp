# Python Bootcamp

A comprehensive Python learning bootcamp with hands-on exercises and examples.

## Structure

- **Section02/**: Hello World basics and fundamental Python concepts
- **Section03/**: Variables, data types, and basic operations
- **Section04/**: Control structures (if statements, loops)
- **Section05/**: Functions and modules
- **Section06/**: Data structures (lists, dictionaries, tuples)
- **Section07/**: File handling and error management

## Setup

1. Clone this repository
2. Create a virtual environment: `python -m venv .venv`
3. Activate the virtual environment:
   - Linux/Mac: `source .venv/bin/activate`
   - Windows: `.venv\Scripts\activate`
4. Install dependencies: `pip install jupyter notebook jupyterlab matplotlib numpy pandas seaborn`

## Running Notebooks

Start Jupyter Lab:
```bash
jupyter lab
```

Or Jupyter Notebook:
```bash
jupyter notebook
```

## Requirements

- Python 3.13+
- Jupyter Notebook/Lab
- Essential data science libraries (numpy, pandas, matplotlib, seaborn)

## Learning Path

Work through the sections sequentially, starting with Section02. Each notebook contains:
- Learning objectives
- Code examples with detailed explanations
- Practice exercises
- Notes and best practices
